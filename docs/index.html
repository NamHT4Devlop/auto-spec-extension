Run Task — 13-Step Pipeline

The core command (Ctrl+Shift+K). Takes one task description and runs a 4-phase AI pipeline: Understand & Plan → Build & Review → Test → Close. Each step calls callCopilot() with the Knowledge Base injected as context. All outputs are saved to a timestamped session folder.
